### 삼항연산자와 범위연산자  
1. 삼항연산자  
    -조건이 딱 2개인 경우(참/거짓)  
    -a > 29 ? print("10대입니다.") : print("20대 이상입니다.")  
2. 범위 연산자  
    -범위를 나타낼 때 사용하는 연산자 예) 1...10, ... 10  
    -한쪽이 열린 연산을 할 때는 띄어쓰기 해야 함.  
    -1..<10, ..>10 과 같은 연산도 가능  
3. 패턴매칭 연산자(~=)  
    -범위연산자와 함께 사용하는 연산자. 해당 범위에 해당되면 참, 아니면 거짓  
    -a...b ~= age
